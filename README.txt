== Description ==

Easy Responsive Test Plugin enables you to check your site responsiveness in your desired window size. This is kind of essential tool for wordpress admins to make sure that their sites.

= Features =

* You can easily test your site on different screen sizes
* Already have some common window sizes
* You can add any custom screen size as per you requirements using settings page.
* You can set custom height and width.
* You can set custom colors for toolbar.


You don't need to re-size browser window every time to test your site.
Just Install ERT and test your site's  responsiveness


For bootstrap shortcodes, check our plugin [Easy Bootstrap Shortcodes](http://wordpress.org/plugins/easy-bootstrap-shortcodes/)

Looking for parallax slider, check this [Easy Wordpress Parallax Slider](http://wordpress.org/plugins/easy-wp-parallax-slider/)

== Installation ==

1.Upload:
Unzip easy-responsive-test.zip file, and upload all files to your blog folder: <root>/wp-content/plugins/easy-responsive-test

2.Activate:
In the admin panel, find Plugins->Plugins,click it, active Easy Responsive Test.

3.Use:
Now you can see the few new ERTP Settings link on left hand side menu, use that link for custom settings and to test site.

It's that simple ... just what the name suggests
